## How to Add Screenshots

1. Run the applications (phone and desktop)
2. Take screenshots of key features
3. Save images to `docs/images/` directory
4. Update README.md to include images:

```markdown
### Android Gateway

![Android Gateway](docs/images/phone-gateway.png)

### Desktop Dashboard

![Desktop Dashboard](docs/images/desktop-dashboard.png)
```

## Image Guidelines

- **Format**: PNG or JPG
- **Size**: Optimize for web (< 500KB per image)
- **Resolution**: Clear and readable (recommend 1280px width)
- **Theme**: Use dark mode screenshots to match the premium dark-mode UI mentioned
- **Annotations**: Optional - highlight key features with arrows or text
